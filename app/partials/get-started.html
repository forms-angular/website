Getting Started

Creating your first application using forms-angular in just a few minutes

These instructions will take you through the process of creating your first forms-angular application, which will consist of an input form for a simple model, a listing showing data already on file, search, menus and a simple report with several features including outputting to PDF. The options will be reached through a simple menu.

If you want a sneak preview of what the input form will look like when you have finished take a look at one of the examples - there is a simple input form here and a more feature-filled one here. Alternatively you could watch the video below that shows the whole process - from starting up a new server to having a running forms-angular app in of an app being generated in just six minutes.

Prepare

(if you need to - you may already have these applications / packages installed)

Please note: the "just a few minutes" in the title starts from after these are installed!

  • Install Node JS and (if it isn't part of the Node install) npm (installation depends on operating system)
  • Install MongoDB (installation depends on operating system)
  • Install Git (installation depends on operating system)
  • Install Yeoman - npm install -g yo
  • Install Bower - npm install -g bower
  • (If you are going to use the file upload plugin with image files) Install imagemagick - npm install -g imagemagick

If you are just having a play you can do all the above on a new Ubuntu server (from Digital Ocean, for instance) you can do the above steps with the following commands

apt-get update && apt-get install -y mongodb git make g++ imagemagick
git clone https://github.com/tj/n.git
cd n && make install  && n stable

Note that you may need sudo before some of the commands above.

Create

At the command line, in the folder you want your application to reside in, do:

npm install -g yo bower generator-fng grunt-cli && yo fng

Follow the onscreen prompts to customise your application. Note that this command must not be run with sudo (see this page for how to install global packages as a user).

Run

grunt serve
